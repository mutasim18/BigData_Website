Intro

In the rapidly growing field of data analytics, selecting the right modelling techniques is important for optimizing predictive performance and gaining insightful business intelligence. This project presents a comprehensive comparative analysis of several advanced regression models developed using different technologies and methodologies. Specifically, we explore models built with Python on Big Query python notebook (utilizing the XGBoost and LightGBM library), and SQL. Using SQL, we came up with three different algorithms: Linear Regression, XGBoost, and LightGBM on Big Query and Azure ML Studio. We will also talk about some of the tools we used while trying to train and deploy one of our models. Tools such as Amazon Web Services (Sagemaker, S3), Google Cloud Platform (Google Cloud Storage, Vertex AI, Big Query), Azure ML Studio and Docker will be talked about. The goal of this analysis is to assess and compare the performance of these models. This includes evaluating their predictive accuracy, computational efficiency, scalability, and ease of implementation in different environments. By doing so, we aim to identify the most effective modelling approach tailored to our dataset on real estate price prediction. We’ll start by outlining the setup, including the data preparation process, model training, and evaluation criteria. Following this, we’ll look into the results of our experiments. This will give us a detailed discussion on the strengths and weaknesses of each model. We’ll also talk about what we tried for the model deployment, what worked and what failed.
