Methodology

Setting up Apache NiFi locally involves a series of steps: Download the Apache NiFi binary package from the official website, choosing the appropriate package for your operating system. Extract the package contents to a directory of your choice. Ensure that Java is installed and configured correctly, setting the JAVA_HOME environment variable if necessary. Launch NiFi by executing the relevant script in the bin folder. Access the NiFi Web UI through a web browser at http://localhost:8080/nifi. For Google Big Query: Access the Google Cloud console and sign in with your Google account. If it's your first time, create a new project. Enable the Big Query API in the "APIs & Services" > "Library" section of the console.
