Methodology

Setting up Apache NiFi locally involves a series of steps:

  1. Download the Apache NiFi binary package from the official website, choosing the appropriate package for your operating system.
  2. Extract the package contents to a directory of your choice.
  3. Ensure that Java is installed and configured correctly, setting the JAVA_HOME environment variable if necessary.
  4. Launch NiFi by executing the relevant script in the bin folder.
  5. Access the NiFi Web UI through a web browser at http://localhost:8080/nifi.

For Google Big Query:

  1. Access the Google Cloud console and sign in with your Google account.
  2. If it's your first time, create a new project.
  3. Enable the Big Query API in the "APIs & Services" > "Library" section of the console.
